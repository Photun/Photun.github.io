Admin portal for PepTunes

Create bands, manage members, assign a drum major, and generate join codes for your players. This portal writes directly to the app's Realtime Database.

Open Admin Portal

How it works

  1. Open Admin Portal and sign up with email/password.
  2. Create a band — this creates `/bands/` in RTDB with `createdBy` set to your uid.
  3. Share the join code with members. When they join in the app, they'll be listed under `/users` with `bandId` set.
  4. In the portal click a member's "Set Drum Major" button to set the drum major (mapped by uid).
